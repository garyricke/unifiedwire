Download & Deliver

Print-ready PDFs are pre-built — just click and send to the printer
Click the PDF button Each sheet card below has a PDF button. Clicking it downloads the latest print-ready PDF — front + back, 8.5″ × 11″, no headers or footers.
Filename includes a version Each PDF is timestamped YYYYMMDD-HHMM (e.g. UWC-03-Building-Lighting-20260429-1430.pdf) so the printer always knows which build they have.
Need a fresh build? Edit any HTML, then run python3 generate-pdfs.py from this folder. Every sheet regenerates with a new timestamp and the buttons below auto-update.
Manual re-print fallback You can also open any sheet via View, press ⌘ P / Ctrl P, choose Save as PDF, and set Paper: US Letter, Margins: None, Scale: 100%, Background graphics: On.
